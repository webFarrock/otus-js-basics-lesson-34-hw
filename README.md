# Домашнее задание курса [OTUS JS Basics](https://otus.ru/lessons/javascript-basic/) - Работа с асинхронными actions в redux

## Основные команды

### Build

```js
npm run build
```

### Dev

```js
npm run dev
```

### Linter

```js
npm run lint
```

### Test

```js
npm run test
```

### Test Coverage

```js
npm run test:coverage
```
